## Features
======

- [IP] Monitor capture (M)

- [W] Rectangular capture axis-locking (M)

======

#### Status Symbols
- W = Waiting
- IP = In Progress
- FC = Feature Completed

- FR = Feature Review
- D = Done

#### Feature Labels
- M = Single monitor
- M+ = Multiple monitors
